# Intro
Service to manage discount codes

# Steps to execute
Python version 3.10
### > virtualenv venv
### > source venv/Scripts/activate or ./venv/Scripts/activate
### > pip install -r requirements.txt
### > python main.py

# Test
### > pytest

# steps to execute using docker container
### > make clean
### > make docker-build
### > make docker-run

To run tests
### > make test
